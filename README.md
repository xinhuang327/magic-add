Magic Add
=========

Perform a magical add operation

## Installation

  `npm install @adrianhuang/magic-add`

## Usage

    var magicAdd = require('@adrianhuang/magic-add');

    var three = magicAdd(1, 2);
  
  
  Output should be 3


## Tests

  `npm test`
